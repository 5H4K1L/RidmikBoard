Welcome to Ridmik Board Web!

Ridmik Board is basically a mod of Ridmik Bangla Keyboard. Ridmik Keyboard is a Bengali and English Keyboard for Android 2.3+. Original app wasn't being updated for long time. So I thought about bringing few changes, and make if more user friendly. That's all!

Development

As I'm a newbie and I don't have the original source code of ridmik Keyboard, I needed to mod the original app and bring changes. This was totally for personal usage. But many of us like me struggled with older emojis and old school themes. So this initiate was to change emojis and themes and make app user friendly! I'm working on this prject and I'll continue till I get bore or Ridmik Dev brings new update! :P
