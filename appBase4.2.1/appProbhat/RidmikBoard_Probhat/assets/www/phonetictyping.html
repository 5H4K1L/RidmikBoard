Typing Practice

চন্দ্রবিন্দু লিখুন qq দিয়েঃ ৎ লিখুন TH দিয়েঃ
চাঁদ = caqqd হটাৎ = hoTaTH
ছোঁয়া = chOqqya পশ্চাৎ = poshcaTH
হসন্ত লিখুন hs দিয়েঃ ং লিখুন ng দিয়েঃ
আল্লাহ্‌ = Allahhs বাংলা = bangla
- সংলাপ = songlap
ঙ লিখুন Ng দিয়েঃ ঞ লিখুন NG দিয়েঃ

বঙ্গ = boNgo ভুঞা = vuiNGa

শ্বাশ্বত = SwaSwoto/SwaSwt অ্যানিমেশন = oZanimeSon
- ব্যবহার = bybohar/bz/Zbohar
ঋ অথবা ঋ-কার লিখতে rriঃ রেফ লিখুন (vowel) + rr + (consonant) দিয়েঃ
ঋণ = rriN কর্ম = korrmo
বৃত্ত = brritto উর্দি = urrdi

ঞ্জ = nj ঞ্চ = nc
